• Bold Fonts and Typography - Bold, high contrast fonts on important data elements help guide the user’s eye. Contrast between interaction areas helps pinpoint focus.
  • User-Modifiable Color Schemes - Color schemes are modifiable not just for user taste, but also to assist in usability in different lighting situations. Easily switching between "dark" and "light" helps the user see important data in any setting.
  • Persistent, Contextual Help Layer - Top-down help systems take the user outside of their work to find help topics. NextGen interfaces rely on contextual help layers which turn on or off, depending on familiarity with the system. Help is always specific to exactly where the user needs it.
