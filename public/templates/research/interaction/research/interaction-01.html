Gaming, Finance and Data Journalism are leading the way in cutting-edge interaction approaches. Developers in these industries are building display layers that allow users to build their own controls, macros and scripts. Users will be able to choose their mode of interaction, and will be able to customize the tools and build on a development layer provided to them by the software.

  • This interaction layer is also flexible enough to allow the user to choose their own input method, such as touch, mouse, gestures or voice.
  • Kinect, Tablet, Smartphone and Desktop Usage no longer need separate applications.

Users don’t want simplicity for simplicity’s sake. They want a consistent visual language with a tiered learning approach so that they can turn on or activate deeper levels of interaction as they learn how to use the system.
